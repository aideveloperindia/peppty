# Complete Guide: Deploy Peppty Technologies Website to GoDaddy

## Prerequisites
- GoDaddy hosting account (with cPanel or File Manager access)
- Domain name configured
- Node.js installed on your local machine

---

## Step 1: Build the Static Export Locally

### 1.1 Open Terminal/Command Prompt
Navigate to your project directory:
```bash
cd C:\Users\DELL\pepptywebsite
```

### 1.2 Install Dependencies (if not already done)
```bash
npm install
```

### 1.3 Build the Static Export
```bash
npm run build
```

This will create an `out/` folder containing all static files.

### 1.4 Verify the Build
- Check that `out/` folder exists
- Inside `out/`, you should see:
  - `index.html`
  - `products/`, `about/`, `legal/`, etc. folders
  - `_next/` folder (static assets)
  - `peppty-logo.png`, `favicon.png`, etc.

### 3.1 Locate Root Directory
- In cPanel File Manager: Look for `public_html/` folder (or `httpdocs/`)

### Option A: Using cPanel File Manager
1. Open File Manager in cPanel
2. Navigate to `public_html/` folder
3. Click "Upload" button
4. Select all files from your local `out/` folder
5. Upload all files (this may take a few minutes)

### Option B: Using FTP
1. Connect via FTP client
2. Navigate to `public_html/` folder
3. Upload all contents from your local `out/` folder:
   - Drag and drop all folders and files
   - Maintain the folder structure
   - Make sure `_next/` folder is uploaded completely

## Step 5: Create/Update .htaccess File

### 5.1 Create .htaccess File
Create a new file named `.htaccess` in your `public_html/` folder with the following content:

```apache
# Enable Rewrite Engine
RewriteEngine On

# Handle Next.js routing - redirect all requests to index.html except files
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ /index.html [L]

# Security Headers
<IfModule mod_headers.c>
  Header set X-Content-Type-Options "nosniff"
  Header set X-Frame-Options "SAMEORIGIN"
  Header set X-XSS-Protection "1; mode=block"
</IfModule>

# Enable Compression
<IfModule mod_deflate.c>
  AddOutputFilterByType DEFLATE text/html text/plain text/xml text/css text/javascript application/javascript application/json
</IfModule>

# Cache Control
<IfModule mod_expires.c>
  ExpiresActive On
  ExpiresByType image/jpg "access plus 1 year"
  ExpiresByType image/jpeg "access plus 1 year"
  ExpiresByType image/gif "access plus 1 year"
  ExpiresByType image/png "access plus 1 year"
  ExpiresByType image/webp "access plus 1 year"
  ExpiresByType text/css "access plus 1 month"
  ExpiresByType application/javascript "access plus 1 month"
</IfModule>

# Prevent directory listing
Options -Indexes
```

### 5.2 Upload .htaccess File
- Upload the `.htaccess` file to `public_html/`
- Make sure it's named exactly `.htaccess` (no extension)

## DNS Configuration Details

### A Record
- Type: A
- Name: @
- Value: Your GoDaddy hosting IP address
- TTL: 600 seconds (or default)

### CNAME Record
- Type: CNAME
- Name: www
- Value: @
- TTL: 600 seconds (or default)

## SSL Certificate Setup

1. Log in to GoDaddy cPanel
2. Navigate to "SSL/TLS Status"
3. Click "Run AutoSSL" or "Install SSL"
4. Select your domain
5. Choose "Let's Encrypt" (free)
6. Click "Install"
7. Wait for installation (usually 5-10 minutes)
8. Enable "Force HTTPS Redirect" in cPanel

## Quick Troubleshooting Guide

### Problem: White screen / blank page
**Check:**
- index.html exists in public_html/
- .htaccess file is present
- File permissions are correct (644)

### Problem: Styles not loading
**Check:**
- _next/static/ folder uploaded completely
- Check browser console for 404 errors
- Verify file paths in HTML

### Problem: Images broken
**Check:**
- Image files exist in public_html/
- Image paths are correct
- File extensions match (.png, .jpg, etc.)

### Problem: Routing not working
**Check:**
- .htaccess file exists and is correct
- RewriteEngine is On
- Apache mod_rewrite is enabled

## Advanced Configuration

### Custom Error Pages:
Create custom 404.html in public_html/:
```html
<!DOCTYPE html>
<html>
<head>
  <title>404 - Page Not Found</title>
</head>
<body>
  <h1>404 - Page Not Found</h1>
  <p>The page you're looking for doesn't exist.</p>
  <a href="/">Go Home</a>
</body>
</html>
```

### Redirects:
Add to .htaccess:
```apache
# Redirect old page to new page
Redirect 301 /old-page.html /new-page.html
```
